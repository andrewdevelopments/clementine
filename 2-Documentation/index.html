Installin WordPress

Installin WordPress is a very simple process and takes less than five minutes to be completed.Many web hosts are offering tools(Bluehost, GoDaddy, mediatemple, etc) for automatic installation. However, if you want to install WordPress by yourself, read the Famous 5 Minutes Installation

Theme Installation

After you downloaded Clementine from Themeforest, inside the zip file you will find a folder named 1-Installation with a zip file clementine.zip

  1. Log in to the WordPress Administration Panels
  2. Select the Appearance panel, then Themes
  3. Select Add New then Upload Theme
  4. Select clementine.zip from 1-Installation folder
  5. Activate

* Note: You should not upload the whole item you have downloaded from Themeforest because you will get an error of broken stylesheet file. Instead locate clementine.zip inside the main file you downloaded

Check video to see how you install Clementine and required plugins

Required Plugins

When you install Clementine you will get a notification message in your WordPress admin informing you about required and recommended plugins. If you are not familiar with plugin management, please check WordPress Codex - Managing Plugins Reference

  • Clementine Shortcodes - Required

    The plugin lets you do nifty things with very little effort. You can add columns, social icons, buttons, dropcaps so please make sure that it is always installed and activated.

  • Vafpress Post Format - Required

    The plugin is a completely additive solution that leverages the default WordPress functionality, while improving the UI and standardizing the names and presentation of custom fields that support the various post formats.

  • MailChimp for WordPress - Recommended

    MailChimp for WordPress helps you add more subscribers to your MailChimp lists using various methods. You can create good looking opt-in forms or integrate with any other form on your site, like your comment, contact or checkout form.

  • Contact Form 7 - Recommended

    Contact Form 7 can manage multiple contact forms, plus you can customize the form and the mail contents flexibly with simple markup. The form supports Ajax-powered submitting, CAPTCHA, Akismet spam filtering and so on

Demo Content

To import demo content into a WordPress site follow these steps:

  1. Log in to that site as an administrator
  2. Go to Administration > Tools > Import
  3. Choose WordPress from the list, install and activate the theme
  4. Run Importer
  5. Click on "Choose File" and browse to select the .xml file from 3-Demo Import Files folder of your downloaded package
  6. Click "Upload file and Import"
  7. You will be asked to map the authors in this export file to users on the blog. For each author, you may choose to map to an existing user on the blog or to create a new user
  8. You will then have the choice to import attachments, so click on the Download and import file attachments button and then click Submit
  9. Posts, Pages, Categories, Tags, Media are imported, so be patient and do not refresh the page untill the "All done. Have fun!" message appears
  10. That's all!

Missing style.css

A common issue that can occur when installing a WordPress themes is "The package could not be installed. The theme is missing the style.css stylesheet." error message being displayed when uploading or activating the theme.

Please make sure you are uploading only the installable theme which is called clementine.zip from 1-Installation folder.

Upload the WordPress file in admin area:
  1. Log in to your WordPress admin area
  2. Browse to Appearance > Themes
  3. Click on "Add New"
  4. Click "Upload"
  5. Browse and select the theme zip file to "Upload"

Upload via FTP
  1. Upload the files to /wp-content/themes/ folder on your server
  2. Login to your WordPress admin area to change the theme

Setting up your theme

By default, WordPress shows your most recent posts in reverse chronological order on the front page of your site.

  • Set up a static page

    Go to Administration > Pages > Add New, add a new Title, add Content you would like to see within the content area of the static page, set a Featured Image if you have one and click Publish, voila we have a brand new Page

  • Set up a post page

    Go to Administration > Posts > Add New, the process is similar with static page, besides we have Categories and Tags

  • Create a new Category

    Go to Administration > Posts > Categories, add a Name and click Add New Category, then you can asign a post to this category from post page

  • Create a new Tag

    Go to Administration > Posts > Tags, add a Name to your tag and click Add New Tag, then you can asign a post to this category from post page

Create a new menu

  • Go to Administration > Appearance > Menus
  • Select Create a new menu at the top of the page
  • Enter a name for your new menu in the Menu Name box
  • Click the Create Menu button

Adding Items to a Menu

  • Locate the pane entitled Pages
  • Select the Pages that you want to add by clicking the checkbox next to each Page's title
  • Click the Add to menu button located at the bottom of this pane to add your selection(s) to the menu that you create in the previous step
  • Click the Save Menu button once you've added all the menu items you want

Google Map

How to enable google map on a Post:

  1. Step 1: Create a Post (e.g. First Post)
  2. Step 2: Search the location you want (e.g. Idaho Falls, Wyoming)
  3. Step 3: Click Search
  4. Step 4: Update

How to enable google map in Hero Section:

  1. Log in to your WordPress admin area
  2. Browse to Clementine > Hero Section
  3. Select Type: Google Map with Posts
  4. Center map by a post, this post will be in the middle of the map, if you leave empty map will not be visible
  5. You can play with options, and you can use snazzy maps to add your own style

Contact Form

To set this up, first you have to install Contact Form 7 plugin, then go to Administration > Contact > Add New

    
    <p>Your Name (required)<br />
        [text* your-name] </p>

    <p>Your Email (required)<br />
        [email* your-email] </p>

    <p>Subject<br />
        [text your-subject] </p>

    <p>Your Message<br />
        [textarea your-message] </p>

    <p>[submit "Send"]</p>

Then fill in your email address and the other details and Save. Once your contact form is set, create a new page for example Contact and add the "Contact Form 7" shortcode [contact-form-7 id="1111" title="Contact form 1"]
